# qiankun

## Project setup
```
cd mic-main
yarn install

cd vite-platform
yarn install

cd vue-platform
yarn install
```

### Compiles and hot-reloads for development
```



cd vite-platform
yarn dev

cd vue-platform
yarn serve

cd mic-main

yarn serve
```

### 复现
see http://localhost:8080/ ，切换两个子应用，观察地址栏的变化
